# 💌 Para María ❤️

Una pequeña página web creada con mucho cariño para una persona muy especial.

Esta web es un recuerdo digital lleno de momentos, fotos y detalles para celebrar nuestro amor. ✨

## 💖 Sobre el proyecto

**Para María** es una página romántica minimalista con:

- 💌 Carta de amor interactiva al entrar
- ❤️ Corazones animados flotando
- ⏳ Contador de días juntos
- 📸 Galería con nuestros recuerdos
- 🎵 Música especial para acompañar la experiencia
- 🌸 Diseño sencillo y elegante
